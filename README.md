# O Simples também é Importante

## Sobre o Projeto
Comecei a estudar Java por conta própria no final do mês passado. Em abril, iniciei a matéria de **Programação Orientada a Objetos (POO)**, onde terei o desafio final de realizar um projeto com **Spring Boot** — algo que nunca tive contato antes. O objetivo será criar um CRUD simples para uma concessionária de carros.

Refletindo sobre como desenvolver esse projeto, resolvi dar um passo atrás e treinar a base antes de partir para o avançado. Percebi que ainda possuo pontos de melhoria em relação à lógica e à sintaxe do Java. 

Sendo assim, decidi focar no básico, pois o simples também é fundamental para um profissional de tecnologia. Iniciei com um projeto clássico: uma **Calculadora funcional**. Este processo foi essencial para:
- Consolidar a lógica de programação;
- Treinar a escrita de código (digitação) em Java;
- Praticar os pilares da Programação Orientada a Objetos.

## Estrutura do Código
Para aplicar os conceitos de POO, separei as responsabilidades em classes específicas para cada operador matemático:
- **Multiplicação** (Tabuada)
- **Adição**
- **Subtração**
- **Divisão**

Durante o desenvolvimento, apliquei estruturas de repetição e condicionais fundamentais:
- `switch case`: Utilizado no `main` para gerenciar as opções do menu e a interação com o usuário.
- `while`: Aplicado para manter o menu ativo até que o usuário decida sair, e também na classe de Multiplicação para gerar a tabuada.
- `if`: Implementado na classe de Divisão para validar e impedir a divisão por zero, respeitando as regras matemáticas.

## Conclusão
Este programa é um material de apoio aos meus estudos. Acredito que, embora muitos saibam criar uma calculadora, consolidar esses fundamentos é o que permite avançar com segurança para frameworks mais complexos como o Spring Boot.
